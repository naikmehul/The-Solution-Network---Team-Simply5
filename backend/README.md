# Grievance Platform - Backend

## Setup
- Install dependencies:
  ```
  cd backend
  npm install
  ```
- Start server:
  ```
  npm start
  ```
- API docs: http://localhost:4000/api-docs

Endpoints:
- GET /api/grievances
- GET /api/grievances/:id
- POST /api/grievances
- PUT /api/grievances/:id
- DELETE /api/grievances/:id
